CSS Classique vs Tailwind

CSS Classique

<div class="carte">
  <h3>Titre</h3>
</div>

/* Dans style.css */
.carte {
  background: white;
  padding: 20px;
  border-radius: 8px;
  box-shadow: 0 2px 4px...;
}

Avec Tailwind

<div class="bg-white
            p-5
            rounded-lg
            shadow">
  <h3>Titre</h3>
</div>

/* Pas de CSS à écrire ! */

Tailwind = plus rapide, tout est dans le HTML. Pas besoin de fichier CSS séparé !

Aide-mémoire Tailwind

Couleurs

  • bg-red-500 Fond rouge
  • text-blue-600 Texte bleu
  • bg-gray-100 Fond gris clair

Espacements

  • p-4 Padding 1rem
  • m-6 Margin 1.5rem
  • px-4 py-2 Horizontal/Vertical

Flexbox

  • flex Active flex
  • justify-center Centre H
  • items-center Centre V
