# Projet Stage 3ème - Création de Site Web

Bienvenue dans ce projet de découverte du développement web !

Ce projet est divisé en **2 sessions** progressives pour apprendre à créer un site web de A à Z.

---

## Vue d'ensemble

| Session | Durée | Thèmes abordés |
|---------|-------|----------------|
| **Session 1** | 2 jours | HTML de base, CSS, structure d'un site |
| **Session 2** | 2 jours | CSS avancé (Flexbox), Tailwind CSS, JavaScript |

## Comment démarrer ?

### Prérequis
- Un éditeur de code (VS Code recommandé)
- Un navigateur web (Chrome, Firefox, Edge...)
- Aucune installation supplémentaire requise !

### Pour le stagiaire

1. **Jour 1-2** : Ouvre `README-SESSION1.md` et suis les instructions
2. **Jour 3-4** : Ouvre `session2/README-SESSION2.md` pour la suite

### Pour le tuteur

Les solutions sont disponibles dans :
- `solutions/` pour la Session 1
- `session2/solutions/` pour la Session 2

## Compétences acquises

### Session 1 - Les fondamentaux
- Structure d'une page HTML (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Balises essentielles (`<h1>`, `<p>`, `<a>`, `<ul>`, `<img>`)
- Navigation entre pages
- Bases du CSS (couleurs, polices, marges)
- Lier un fichier CSS externe

### Session 2 - Niveau intermédiaire
- **Flexbox** : mise en page moderne
- **Tailwind CSS** : framework CSS utility-first (via CDN)
- **JavaScript** :
  - Sélectionner des éléments (`getElementById`, `querySelector`)
  - Réagir aux événements (`addEventListener`)
  - Modifier le DOM (`textContent`, `classList`)
  - Variables, conditions, boucles
  - Créer un quiz interactif

## Conseils pour le tuteur

1. **Laisser chercher** : Ne donnez pas la réponse trop vite, guidez plutôt
2. **Tester souvent** : Encouragez à sauvegarder et rafraîchir le navigateur régulièrement
3. **Console F12** : Montrez comment utiliser la console pour débugger le JavaScript
4. **Personnalisation** : Encouragez à modifier les couleurs, textes, et à s'approprier le projet
5. **Solutions** : Utilisez-les uniquement si le stagiaire est vraiment bloqué
