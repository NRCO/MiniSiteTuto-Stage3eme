# Exercice Autonome - Site de Recettes

Tu as appris les bases du HTML et du CSS. Maintenant, c'est à toi de jouer !

---

## Objectif

Créer un **mini site de recettes** avec :
- Une page d'accueil listant plusieurs recettes
- Une page de détail pour une recette
- Un style CSS personnalisé

---

## Ce que tu dois faire

### 1. Page d'accueil (`index.html`)

- Lier le fichier CSS
- Créer une navigation avec 2 liens
- Ajouter un header avec le titre du site
- Afficher 3 recettes (chacune dans un `<article>`)
- Chaque recette doit avoir : un titre, une description, un lien
- Ajouter un footer

### 2. Page de détail (`recette.html`)

- Même navigation que l'accueil
- Header avec le nom de la recette
- Une image (utilise Unsplash ou Pexels)
- Liste des ingrédients (`<ul>`)
- Étapes de préparation (`<ol>`)
- Lien retour vers l'accueil
- Footer

### 3. Feuille de style (`style.css`)

Ajoute du style pour :
- `body` : police, couleur de fond
- `nav` : fond coloré, liens blancs
- `header` : couleur de fond, texte centré
- `main` : largeur max 700px, centré, fond blanc
- `article` : espacement, bordure
- Titres, paragraphes, liens
- `footer` : comme le nav

---

## Indices

| Élément | Balises utiles |
|---------|----------------|
| Navigation | `<nav>`, `<a href="">` |
| En-tête | `<header>`, `<h1>` |
| Contenu | `<main>`, `<article>` |
| Listes | `<ul>`, `<ol>`, `<li>` |
| Image | `<img src="" alt="">` |
| Lier CSS | `<link rel="stylesheet" href="style.css">` |

---

## Critères de réussite

- [ ] Les 2 pages s'affichent sans erreur
- [ ] La navigation fonctionne entre les pages
- [ ] Le CSS est bien lié et appliqué
- [ ] La mise en page est différente du projet gaming
- [ ] Le code est propre et indenté
