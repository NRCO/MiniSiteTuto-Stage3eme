# Dossier Solutions

Ce dossier contient les fichiers **terminés** pour que le tuteur puisse :
- Vérifier le travail du stagiaire
- Aider en cas de blocage
- Montrer le résultat attendu

## Fichiers inclus

- `index.html` - Page d'accueil complète
- `article1.html` - Article GTA 6 complet
- `article2.html` - Article PC Gaming complet
- `article3.html` - Article Minecraft complet

## Comment utiliser

1. **Ne pas montrer ces fichiers au stagiaire** au début !
2. Les utiliser seulement si le stagiaire est vraiment bloqué
3. Comparer avec le travail du stagiaire pour validation

## Note

Les liens CSS pointent vers `../style.css` pour fonctionner depuis ce sous-dossier.
Pour tester, ouvrir directement ces fichiers dans le navigateur.
